About

Overview

netloom is a tool for operators and automation engineers to simplify day-to-day tasks. HPE Aruba ClearPass is a bundled plugin and the CLI is organized around operator-friendly workflows.

  • Shared command model for list, create, update, delete, diff, and copy operations
  • Workflows for development staging -> production deployment with dry-run planning
  • Cached discovery, context-aware help, and shell completion from the CLI
  • JSON, CSV, and raw output paths for reuse and automation
Install

Install And First Run

Install the package, load the plugin, choose a profile, refresh the cache, and explore directly from the terminal. The first cache update enables richer help, completion, and live module and service discovery. 

pip install netloom-tool
netloom load clearpass
netloom server use <profile>
netloom cache update
netloom identities endpoint list --limit=10
